</h3>

Inspired by Claude Code — an intelligent coding assistant that reads, writes, and executes code in your terminal.

- 🤖 Agentic Loop with streaming output & thinking process
- 🛠️ 7 built-in tools + MCP protocol for unlimited extensions
- 🔒 7-stage execution pipeline with 4 permission modes
- 🎨 Ink-powered terminal UI with Markdown rendering & 5 themes
- 🧩 Slash commands, skills, hooks — extend with Markdown files
- 📖 [17-chapter tutorial](https://kkkhs.github.io/ClawdCode/) — build one from scratch

```bash
npm install -g clawdcode
```
